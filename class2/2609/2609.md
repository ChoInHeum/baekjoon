# 최대공약수(GCD), 최소공배수(LCM)
## 사용한 알고리즘
- **유클리드 호제법**

### 알고리즘 설명
- 두 양의 정수 $a$와 $b(a>b)$가 있을 때, $a$를 $b$로 나눈 나머지가 $r$이면 $a$와 $b$의 최대공약수는 $b$와 $r$의 최대공약수와 같다는 것이다.

### 알고리즘 예시
- $24(a)$와 $18(b)$의 최대공약수

|   |GCD| a | b | a%b |
|:---:|:---:|:---:|:---:|:----:|
|1회|GCD(24, 18)|24|18|6|
|2회|GCD(18, 6)|18|6|0|

- $24$와 $18$의 최대공약수(GCD)는 $6$이다.
- 또한, 최소공배수(LCM)는 $a \times b \; \div \; GCD(a, b)$이므로 $24$와 $18$의 최소공배수(LCM)는 $72$이다.

## 코드 (Python)
### 내 코드
``` Python
import sys

input_value1, input_value2 = map(int, sys.stdin.readline().split())
GCD, LCM = 0, 0

if (input_value2 > input_value1):
    temp = input_value2
    input_value2 = input_value1
    input_value1 = temp

a, b = input_value1, input_value2

while True:
    if (a % b == 0):
        GCD = b
        break

    temp = a % b
    a = b
    b = temp

LCM = int(input_value1 * input_value2 / GCD)

print(GCD)
print(LCM)
```
### 최적화 코드(AI)
``` Python
import sys

a, b = map(int, sys.stdin.readline().split())
x, y = a, b

while y:
    x, y = y, x % y

gcd = x
lcm = a * b // gcd

print(gcd)
print(lcm)
```
